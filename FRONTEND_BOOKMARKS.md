## Why Bookmark Docs You Already Use
- Docs help when you know the tool exists, but need the exact API, syntax, pattern, or updated recommendation.
- Next.js changes quickly, so the docs are often the fastest way to confirm the current pattern.
- Tailwind docs are useful for finding the exact utility class, responsive variant, state modifier, or layout trick without guessing.
- MDN is the long-term fallback when the problem is really HTML, CSS, JavaScript, or browser behavior rather than the framework.
